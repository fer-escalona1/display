Display: lo que mostramos en la pantalla

Display None

Este tipo de display nos permite dejar de visaulizar contenido dentro del sitio.

Cuando lo usamos, las etiquetas dejan de verse pero el código sigue ahí.

Display Block

El elemento genera un cuadro de elemento del bloque.

Significa que mi bloque (recuerda que todo en HTML es un bloque), ocupa un espacio disponible de izquierda a derecha. o todo el renglón.

Display InLine

El elemento genera uno o más cuadros de elemento en línea.

Significa que el contenedor o bloque se adapta al ancho de su contenido hijo.

Display Block-InLine

Si ponemos tamaño a un Display Block, funciona, pero sigue ocupando todo un renglón.

Si le ponemos tamaño a un display inLine ignora las propiedades de alto y ancho.

Esto puede ser problemático. Pero con InLine-Block podemos fusionar ambas.
